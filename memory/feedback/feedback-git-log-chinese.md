## 知識

- [臨] Git commit message 一律用繁體中文寫說明。conventional-commits prefix（feat(scope): / fix(scope): / test(scope): 等）與 Co-Authored-By footer 保留英文（格式要求），但 summary line + body 全部中文，不寫「Key findings:」「Background:」等英文段落標題。
- [臨] 來源：使用者 2026-04-16 在 V4.1 P4 session 明確提醒「上傳GIT也是請注意用中文寫Log喔」，「也」字表示是既有既定偏好。
- [臨] 正確範例：prefix 英文 + body 中文，附註用「重點發現」「背景」「附註」中文小標，不混英文小標題。
