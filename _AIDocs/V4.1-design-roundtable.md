## Phase A — 10 大師 Drafting 摘要

### 人文專精
- 決策三要素：①未來行動 ②排他性 ③承諾強度，缺一不抓
- 20 組該抓/不抓對照例（含中文婉轉語、情緒混合句）
- agency 紅線：白名單 scope，情緒/身體/人際永不入庫

### UX 學
- Surface 策略矩陣：萃取靜默、pending 提示（節流 ≥3 或 ≥6h）、衝突 systemMessage
- 一鍵 reject `/memory-undo`，弱透明性 `/memory-peek`
- 盲測：N=1 誘餌題 + Likert ≥ 4

### 程式專精
- 5 phase：flag infra → UserPromptSubmit detector → Stop async worker → chaos test → 跨平台
- state_transaction 防 race；hot_cache 不碰；fail-open；orphan PID 清理
- 測試：pytest unit/integration/regression/chaos

### AI 專精
- 三層級聯：rule → qwen3:1.7b(L1, think=false) → gemma4:e4b(L2, 仲裁)
- L1 few-shot 含時間副詞邊緣例；num_predict=80；T=0
- scope/audience 單階段推斷省 tok；invalid JSON salvage + 不重試

### 原子記憶專精
- scope decision tree：「我」→ personal、「團隊」→ shared、「{role}組」→ role
- 短句 dedup embedding 0.88 fallback；trigger LLM 抽 noun/verb
- Pending review 流量控制：scope=shared+audience=decision 才進 queue

### 實作大師
- MVP = detector + extractor 兩件做透；能砍 8 項（taxonomy/dashboard/SVN/ensemble/promotion/回填/stance 全自動/UI feedback）
- 黑啟動 → 灰度 → whitelist → 正式；超 7d = scope 爆
- 複用 ollama 通道 + atom_write MCP；新建 detector + prompts

### 精省 token 大師
- L0 過 88%+ 是達標關鍵；prompt cache 友好設計
- Session budget tracker 必建；Mem0 SOTA 1.8K tok/conv（V4.1 的 9 倍）
- Token 估算表：30 prompts × 10% 決策密度 → 168 tok/session

### 語意理解大師
- 四層 features：詞彙(信號詞權重) → 句法(主詞+情態+動作) → 語用(陳述 vs 疑問) → 上下文(topic)
- Hybrid threshold routing：conf ≥ 0.92 直寫 / 0.70-0.92 review / < 0.70 丟
- MVP = Phase A(lexical)+B(stance) → D(threshold)，C(context) 留 v4.1.1

### 資訊整合 #1（Prior Art）
- ChatGPT Memory：顯式+自動雙通道；€15M GDPR 罰款；precision no public data
- Mem0(ECAI 2025)：LOCOMO 66.9%（SOTA）；1.8K tok/conv；生產環境 10134 筆 97.8% junk
- Zep/Graphiti：DMR 94.8%；時序 KG 收斂設計同 V4 三時段
- Claude Memory Tool(2025-09)：agent CRUD + client-side FS
- 3 大過度幻想方向：LLM zero-shot ≠ 0.92 / 使用者不會主動 reject / 盲測需 4-6 週

### 資訊整合 #2（NLP Benchmark）
- 中文短文 intent：CLUE-IFLYTEK Acc~60%；gemma-3-1b-it F1 0.36 / Qwen3-4B 0.65
- Stance detection：SemEval macro-F1 0.678；中文 Weibo 0.71；CoS+Mistral-7B 76.43
- Mem0 production audit: #4573 97.8% junk、#4540 silent false-extraction
- 3 大過度樂觀預期：small LLM zero-shot 不可能 P 0.92 / AL 不解 subjective label / two-stage 典型 P 0.85-0.90

## Phase C — 8 大師 Validation（視角調換）

**Verdict 統計：8/8 iterate、0 ship、0 reject**

| 原視角 | 扮演視角 | 關鍵反對點 |
|---|---|---|
| 人文 → 精省 token | 婉轉語跨 turn token 翻倍；state 膨脹 parse 稅；skill 注入隱性 cost |
| UX → 程式 | Haiku 零實作基礎（無 SDK）；regression fixture 不存在；server.js 重啟廣播 |
| 程式 → AI | L1 conf 不可靠（小模型 ECE 0.15-0.25）；Haiku prompt 未校準；cache 中文命中假設 |
| AI → 原子記憶 | Source-turn-id 破 SPEC = V4.2；dedup 否定詞誤判；Confidence 永凍 [臨] |
| 原子記憶 → UX | 首次發現延遲信任崩盤；婉轉確認=偷記；explain ROI 低應砍 |
| 實作 → 語意 | N=1/200 字切斷 stance；雙重確認≠stance；歷史不回填→首日雞肋；mega decisions.md |
| 精省 token → 人文 | 混合句無演算法；personal 隱私外洩路徑；情緒承諾24h 冷卻；informed consent |
| 語意 → 實作 | 7d→12-14d 低估；explain 砍掉；extract-worker 仿坑；server.js 走 extra_metadata |
