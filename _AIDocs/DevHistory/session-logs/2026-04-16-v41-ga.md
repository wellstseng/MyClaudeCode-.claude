## 摘要

清除 rc2 GA blocker（整合測 Recall=0）。三處根因修復 + feature flag 打開。

## 三處根因

### (1) `tools/ollama_client.py::_request_with_failover`

當 `explicit_model` 與 backend `llm_model` 不符時 **skip without `_record_failure`**，避免單次 404 毒化 health_cache 60s 使後續全 silent return。

### (2) `hooks/user-extract-worker.py::_call_l1` fallback

加 fallback pattern：優先 qwen3:1.7b 快速路徑，失敗則 fallback 到 backend 預設 model，同 `_call_l2`。

### (3) prompt regex bug 修復

`_load_l1_prompt`/`_load_l2_prompt` 的 `re.search(r"\`\`\`\n(.*?)\`\`\`")` 對 L2.md 兩個 code block 會匹配到 Output Schema 結束到 Prompt 開始之間的 94-char 空白段（真正 prompt 從未送進 model）→ 改用 `re.findall(r"\`\`\`(?:\w+)?\n(.*?)\`\`\`")` 取最後一個 block。
